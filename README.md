A freelancing-style web app built with Node.js, Firebase, Stripe, and MongoDB. Supports three roles — Admin, Buyer, and Worker — with secure authentication, task management, and payment integration.

The frontend is a dynamic single-page application built to showcase my work and skills. Key features include:
*   **Core Tech:** Built with **React** and **Vite** for a fast and modern development experience.
*   **Styling:** Likely uses **Tailwind CSS** (common for portfolios) for a responsive and clean UI, though specifics are in the submodule.
*   **Functionality:** Fetches project data and other dynamic content from the backend API to keep the portfolio easily updatable.

### ⚙️ Backend Overview
The backend serves as the API and content manager for the portfolio. Its main responsibilities are:
*   **Core Tech:** A **Node.js** server using the **Express.js** framework.
*   **Data Management:** Provides RESTful API endpoints, likely to serve project details, blog posts, or other portfolio content from a database (specifics within the backend submodule).
